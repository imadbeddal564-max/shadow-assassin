# 🎮 Shadow Assassin - قاتل الظلال

لعبة Stealth تكتيكية احترافية مطورة باستخدام Unity و C#

![Game Banner](https://via.placeholder.com/1200x400?text=Shadow+Assassin)

## 📖 نظرة عامة

**Shadow Assassin** هي لعبة stealth من منظور الشخص الثالث (Third Person) حيث تلعب دور عميل سابق تم خيانته، تتسلل إلى منشآت أمنية لسرقة ملفات سرية واغتيال أهداف محددة.

### المميزات الأساسية:
- 🕵️ تسلل احترافي مع نظام رؤية متقدم
- 🤖 ذكاء اصطناعي ديناميكي للأعداء
- 📊 نظام صوتي واقعي (الخطوات، التنفس، الإنذارات)
- 🎮 حركة سلسة وكاميرا احترافية
- 🗺️ مستويات معقدة متعددة الطرق
- ⚙️ نظام الاختراق الإلكتروني
- 💾 نظام حفظ وتحميل

---

## 🚀 البدء السريع

### المتطلبات:
- **Unity 2022 LTS** أو أحدث
- **Visual Studio** أو **Rider**
- **Git**

### الخطوات:
```bash

git clone https://github.com/imadbeddal564-max/shadow-assassin.git

# 2. فتح المشروع في Unity
# افتح Unity Hub → Add → اختر المجلد

# 3. انتظر استيراد الملفات (قد يأخذ وقتاً)
```
